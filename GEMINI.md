# Orbito Project Guidelines

## Critical Workflows
- **Permission First**: ALWAYS Explaining your modifications about codes first, and ask for explicit user permission before making any code modifications or file changes.
- **Auto-Start**: At the beginning of every session or when requested to work on the frontend, ensure that `npm start` is running in the `frontend` directory. If it's not active, start it as a background process.
- **Environment:**
  - Frontend: `frontend/`
  - Backend: `backend/`

## Preferences
- Use background processes for servers.
- Prioritize surgical edits in `App.js`.
